# Vue App CRUD

## Instalar Vue globalmente
`npm install -g @vue/cli`

## Criar novo projeto
`vue create crud-vue`

## Instalar dependencias
`npm install`

### Executar projeto
`npm run serve`

## Gerar para produção
`npm run build`
